El Mandamás

This game was developed for the Ludum Dare 38 online game jam, where participants have up to 72 hours to create a working prototype. The theme was "Small world" so I decided to make a game about Panama's geography since it's a very small country.

The game also got featured in "The best free games of the week" post by Rock Paper Shotgun which can be accessed clicking here .

Moa VR

Moa VR is an HTC Vive game that features New Zealand's exctinct, wingless bird: the Moa. In the game, you are the Moa's keeper and get to learn about it as it progresses. The gameplay is very short yet engaging. I was in charge of bringing UX to life, since most common practices are to be changed when developing for Virtual Reality.

NZGDC17

Ink Wars

I worked as a Unity Developer for this game, developing the Network Versus mode through an MVC architecture. We had to create an asset bundle system that didn't slow down the game due to the fact that it had 27 characters at the time. My main tasks lied inside the Visual & UI/UX environments.
